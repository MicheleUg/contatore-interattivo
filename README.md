
# Contatore Interattivo

# Descrizione
Questo progetto è un semplice contatore realizzato con HTML, CSS e JavaScript.  
Permette di incrementare, decrementare e resettare un valore visualizzato sullo schermo.  

Progetto sulla manipolazione del DOM e nella gestione degli eventi in JavaScript.

---

# Funzionalità
- Incrementa il contatore
- Decrementa il contatore
- Resetta il contatore a zero
